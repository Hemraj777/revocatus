Revocatus

Clean up your Facebook likes, reactions, and interactions.

Recommended Browser Extension

Works on Desktop Chrome and Android (Kiwi Browser / Firefox).

Download Extension
  1. Download and unzip the file.
  2. Desktop Chrome: Go to chrome://extensions, enable Developer mode, click Load unpacked, select the extension folder.
  3. Android (Kiwi): Go to kiwi://extensions, enable Developer mode, load unpacked.
  4. Go to Facebook Activity Log → Likes and Reactions, click the Revocatus icon, then Start.

Bookmarklet (Alternative)

No installation needed — works in any browser.

Long-press to bookmark
  1. Long-press the button above → Copy link address.
  2. Bookmark this page, then Edit the bookmark and paste the copied URL.
  3. Go to Facebook Activity Log → Likes and Reactions, tap the bookmark.
  4. For best results, enable Desktop site in Chrome menu first.
